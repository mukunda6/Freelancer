# **App Name**: Freelancer Hub

## Core Features:

- User Authentication: Secure user login and registration with Firebase Authentication.
- Profile Management: Users can create and edit their profiles, including skills, experience, and portfolio.
- Project Listings: Freelancers can browse and apply for projects posted by clients.
- Client Project Referral System: A feature where verified clients can directly refer new projects to verified freelancers within the platform, bypassing the general marketplace.
- Integrated Project Workspace: Provides a dedicated dashboard where the freelancer can track project milestones, invoice status, and client communication, keeping all data centralized.
- Automated Profile Translation: Translate user-generated content (profiles, project descriptions) using Google Translate API.
- Profile Verification: Integrate with Stripe Identity to verify freelancer identities, increasing client trust.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey professionalism and trust.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Teal (#009688) to highlight key actions and information.
- Body font: 'Inter', a sans-serif font providing a modern, machined, objective, neutral look suitable for body text.
- Headline font: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel for the headlines.
- Use consistent and professional icons to represent different project categories and actions.
- A clean and responsive layout that adapts to different screen sizes.
- Subtle animations to enhance user experience, such as loading indicators and transitions.
